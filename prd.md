## 1. 产品概述

AI 寻标 Agent 是一套基于大模型的招标商机智能平台：自动采集多个招标采购平台的公告，利用 AI 完成信息提取、智能分类、企业能力匹配、风险分析与商机推荐，并通过多渠道推送通知，帮助企业快速发现高价值项目、提高投标效率、降低人工筛选成本。

产品不是传统的招标信息聚合网站，而是一名帮助企业寻找商机的 **AI 智能销售助手**。

---

## 2. 产品定位

把企业销售从**"人工找项目"升级为"AI 主动发现商机"**。与传统招标信息网站的本质区别：

| 维度 | 传统招标信息网站 | AI 寻标 Agent |
| --- | --- | --- |
| 信息获取 | 用户自行检索、订阅关键词 | 系统主动采集、主动推荐 |
| 内容处理 | 展示公告原文与附件链接 | 理解公告：字段提取、分类、附件结构化 |
| 筛选方式 | 关键词过滤 | AI 语义匹配企业能力画像 |
| 判断建议 | 无 | 匹配度评分 + 风险分析 + 参与建议 |
| 推荐结果 | 人人相同 | 每家企业基于自身画像各不相同 |

---

## 3. 产品目标

替代销售人员每天浏览大量招标网站、人工筛选项目、判断是否适合投标等重复性工作，具体实现：

- **自动获取**多个平台招标信息 —— 无需人工巡站
- **AI 自动理解**招标内容 —— 含 PDF/Word 附件的深度解析
- **自动匹配**企业能力 —— 基于企业专属能力画像
- **自动推荐**高价值项目 —— 星级分级，宁缺毋滥
- **自动推送**商机通知 —— 多渠道，日报 + 重大项目即时提醒
- **沉淀企业专属商机数据库** —— 形成可积累的企业知识资产

## 5. 产品功能

### 5.1 招标信息采集

系统自动采集多个招标采购平台，支持定时采集与 7×24 全天候自动运行，采集源可配置、可监控（源站改版自动告警）。所有公告均注明原始出处链接，可跳转源站核实。

支持的平台类型：

- 国家政府采购平台
- 公共资源交易平台
- 各省市采购网
- 医院采购平台
- 高校采购平台
- 企业采购平台
- 能源行业采购平台
- 电网采购平台
- 用户自定义网站

### 5.2 AI 信息解析

系统自动解析公告内容，提取以下字段：

| 字段 | 说明 |
| --- | --- |
| 项目名称 | 归一化后的项目名称 |
| 招标单位 | 采购人 / 招标人名称 |
| 发布时间 | 公告发布日期 |
| 投标截止时间 | 关键时间节点，驱动时效提醒 |
| 项目预算 | 金额归一化（万元/元统一） |
| 所属行业 | 项目所属行业 |
| 地区 | 省 / 市级归属地 |
| 产品类别 | 采购的产品类型 |
| 服务类型 | 采购的服务类型 |
| 联系方式 | 采购联系人及电话 |
| 附件信息 | 附件清单及解析状态 |
| 招标要求 | 资质、业绩、技术等核心要求摘要 |

对 PDF、Word 等附件自动识别正文内容（扫描件走 OCR）并纳入结构化分析。

**质量保障**：所有 AI 提取字段可追溯至公告原文（附依据片段）并带置信度，低置信度字段支持后台人工复核修正。

### 5.3 智能分类

AI 自动完成项目分类，用户无需建立和维护复杂关键词。分类体系示例（可按行业扩展配置）：

| 大类 | 子类 |
| --- | --- |
| IT 类 | GPU、服务器、存储、网络设备、云平台 |
| 软件类 | OA、ERP、数字政府、数据平台 |
| 工程类 | 安防、弱电、综合布线 |

### 5.4 企业能力匹配

建立企业能力画像，包括：**产品、服务、行业、区域、资质、成功案例、合作品牌**。画像支持后台表单维护，也支持上传公司介绍文档由 AI 自动抽取初始化。

AI 基于画像自动计算每个项目的匹配度。示例：

| 项 | 内容 |
| --- | --- |
| 项目 | GPU 集群采购 |
| 企业能力 | 主营 GPU 服务器，具备相关行业案例 |
| 匹配度 | 98% |
| 推荐等级 | ★★★★★ |

### 5.5 商机推荐

每天自动分析全部采集的招标信息，只推荐真正值得关注的项目。示例（某日）：

- 今日共采集：**862 条**
- AI 推荐：**18 条**

| 推荐等级 | 数量 |
| --- | --- |
| ★★★★★ | 5 条 |
| ★★★★☆ | 8 条 |
| ★★★☆☆ | 5 条 |

### 5.6 风险分析

AI 对推荐项目逐项进行风险判断：

- 是否限制品牌
- 是否存在排他性条件
- 是否需要特殊资质
- 是否预算不足
- 是否竞争激烈
- 是否存在废标风险

每项风险给出**判断依据（引用公告原文）与严重程度**，并综合输出**参与建议**：建议参与 / 谨慎参与 / 不建议参与。

### 5.7 消息通知

支持多渠道推送：**邮件、企业微信、钉钉、飞书、Web 站内消息、微信**。

- **每日商机日报**：汇总昨日采集总量与推荐列表（按星级分组）；
- **重大项目即时提醒**：满足订阅规则（如星级阈值、预算门槛）立即推送；
- 推送规则（星级阈值、行业/区域过滤、静默时段）可按用户配置。

> 注：个人微信推送依赖微信公众号（服务号）模板消息资质，V1 以邮件、企业微信、钉钉、飞书、Web 消息先行（已确认），个人微信在资质就绪后接入。

### 5.8 数据查询

支持多维度筛选与 AI 自然语言搜索：

- 关键词 / 行业 / 地区 / 预算 / 发布时间 / 采购单位 / 项目类型筛选
- AI 自然语言搜索，例如：

> "查找江苏省预算超过300万的 AI 项目。"

## 7. 产品特点

| 特点 | 说明 |
| --- | --- |
| **自动化** | 无需人工每天浏览网站，7×24 自动采集与分析 |
| **智能化** | 不仅获取公告，更理解公告：提取、分类、结构化 |
| **精准推荐** | 不是关键词搜索，而是 AI 对企业能力画像的语义匹配 |
| **企业专属** | 每家企业拥有独立知识库与画像，推荐结果完全不同 |
| **结论可追溯** | AI 提取与风险判断均附原文依据，可人工复核 |
| **可扩展** | 支持新增招标平台、行业分类与 AI 模型 |

## 9. 产品价值

**对于企业：**

- 降低每天人工寻找项目的成本
- 提高项目发现效率与销售线索质量
- 减少无效投标，增加中标机会

- 统一管理商机，形成企业知识资产
- 建立数据分析能力，实现销售数字化

## 11. 产品愿景

AI 寻标 Agent 不只是一个招标信息平台，而是企业销售团队的智能商机助手。

通过持续采集市场信息、理解招标需求、匹配企业能力、推荐高价值项目，帮助企业从"人工找项目"升级为"AI 主动发现商机"，最终构建覆盖**"发现商机 — 分析商机 — 辅助投标 — 沉淀知识"**的智能化业务平台。
