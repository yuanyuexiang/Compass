01市场机会

一个 30 万亿级、刚性增长、且正在数字化的市场

招投标是中国经济的"毛细血管"——政府采购、工程建设、国企央企采购都必须公开招标。这个市场规模巨大、政策刚性、且信息正快速电子化公开，为 AI 采集与匹配提供了理想土壤。

0

招标采购总规模（2025E）

2024 年约 34.74 万亿，其中工程招标 26.82 万亿、政府公开采购 2.49 万亿、其他 5.43 万亿。年均复合增速 20%+。

0

全国政府采购规模

财政部披露 2022 年约 3.66 万亿元，占 GDP 约 3%，近年稳定在 3.6–3.9 万亿，刚性可持续。

百亿级

标讯信息服务细分（SOM）

数十万家企业为"招投标信息服务"付费，客单价 2000–8000 元/年，构成百亿级、可被 AI 重做一遍的存量市场。

TAM → SAM → SOM：广义招标采购 30 万亿+（TAM，我们服务的交易底盘）→ 有投标需求、需盯标讯的企业市场（SAM）→ 当前为标讯付费的数十万家企业、百亿级订阅市场（SOM，我们直接替代的蛋糕）。司南不抽交易佣金，而是把"帮企业找到并赢得这些交易"做成订阅服务。
政策顺风：《深化公共资源交易平台整合共享指导意见》推动交易全流程电子化、一网通办；财政部推进政府采购信息"应公开尽公开"。信息越电子化、越公开，合规采集的数据基础越好、监管风险越低——这是 AI 寻标的时代前提。

数据来源：智研咨询/华经产业研究院（招标采购规模）、财政部 mof.gov.cn 与中国政府采购网 ccgp.gov.cn（政府采购规模）、国办《公共资源交易平台整合共享指导意见》、工信部中小企业统计。标讯细分市场为基于竞品客单价与付费企业量的行业估算。

02核心痛点

信息过载，是每个投标企业的日常黑洞

全国 90%+ 企业是中小企业，是投标主力，却极少有专职标讯团队。他们每天面对的是一个无法用人力解决的信息问题。

🌊

信息过载

全国每天新增数万条公告，分散在数百个交易平台。人工逐条筛选如同大海捞针，错失商机成为常态。

⏳

人力昂贵且低效

一个销售/商务人力成本 8–15 万元/年，却把大量时间耗在"翻网站、看公告"这类低价值检索上。

🎯

匹配靠关键词，漏标错标

现有工具按关键词订阅，"智能化"搜不到"弱电工程"，语义不通导致既漏真商机、又推一堆噪音。

⚠️

风险后置，无效投标

行业中标率仅 10–30%。排他条款、资质门槛、预算不足往往投标后才发现，大量精力打了水漂。

数据来源：工信部中小企业统计（企业结构）、行业调研（中标率、人力成本区间为行业普遍估算）。

03解决方案

从"推信息"升级到"给决策"

司南不只是把公告推给你，而是像一个不知疲倦的资深商务：读懂每条公告、判断值不值得投、提示有哪些坑，把结论连同原文依据一起送到销售面前。

🛰️

多源合规采集

官方源自采（政府采购网、公共资源交易中心），httpx + Playwright 双引擎，突破 JS 动态渲染与反爬，只采公开源、不碰聚合站。

🧠

AI 语义理解

DeepSeek 大模型十二字段结构化提取 + 向量化语义匹配，"智能化"能匹配到"弱电/信息化/集成"，告别关键词漏标。

📊

风险评分卡

0–100 匹配分 + 五星推荐 + 参与建议 + 六维风险扫描，每项都附上公告原文依据（evidence），AI 结论可追溯、可信。

一句话：竞品回答"有哪些标"，司南回答"你该投哪个标、为什么、有什么坑"。这就是从信息服务到决策服务的代差。
04产品与技术

已跑通的全链路，不是 PPT 里的构想

采集 → 清洗 → AI 提取 → 向量化 → 三级漏斗匹配 → 风险分析 → 多渠道通知，端到端状态机驱动，每条公告只处理一次、全租户共享。

🔍

三级漏斗匹配引擎

规则硬过滤（零成本淘汰 70%）→ 向量粗排 → LLM 精排评分。层层收窄，让昂贵的大模型只算"漏斗尖"，兼顾精度与成本。

🛡️

六维风险扫描

品牌限制 / 排他条件 / 资质缺失 / 预算不足 / 竞争激烈 / 废标风险，逐项给命中依据与严重度，投标前就看清。

🏢

多租户隔离架构

公共数据层（采集/提取，只算一次）与租户层（画像/匹配，各算各的）分离——这是成本护城河的技术根基。

🔔

多渠道触达

站内信 + 邮件 + 企业微信 / 钉钉 / 飞书，按订阅规则推送每日商机日报与高星即时提醒，商机不落地不罢休。

05商业模式

高毛利订阅制 SaaS，每多一个客户几乎纯是利润

技术架构里藏着商业模式的杀器：公共层把每条公告只处理一次、全租户共享；每新增一个客户，只多算一点"精排增量"，边际成本趋近于零。前期摊平固定成本后，规模越大、毛利越高。

💰

成本结构杀器

公共层 AI 提取日成本仅约 ¥6–10，全体租户共享。新增租户边际成本近零 → 天然高毛利、强规模效应。

07竞争格局

老玩家卖信息，司南卖决策

剑鱼、千里马、比地等已占据市场，但它们仍停留在"关键词订阅 + 信息聚合"，且普遍依赖爬取聚合数据。这既是它们的护城河，也是它们的软肋。

维度传统标讯（剑鱼/千里马/比地）司南 · AI 寻标
匹配方式关键词 / 行业地区过滤，漏标错标语义向量匹配，理解"智能化=弱电=集成"
交付内容推信息（有哪些标）给决策（该投哪个、多少分、有何风险）
风险分析无六维风险 + 原文依据 + 参与建议
数据来源多爬取聚合平台，有合规风险只采官方公开源，合规可持续
成本结构人工 + 爬虫维护，边际成本高AI 中台共享，边际成本近零
合规即壁垒：数据抓取不正当竞争已有明确判例——大众点评诉百度判赔 300 万、新浪微博诉脉脉判赔 200 万、"车来了"数据案主管被追刑责。竞品普遍爬聚合站，司南只采官方公开源，在监管收紧的趋势下是可持续的结构性优势。

竞品信息来源：jianyu360.com、qianlima.com、bidcenter.com.cn 公开资料；判例来源：中国裁判文书网及法律媒体公开报道。

08护城河与数据飞轮

越跑越深的数据飞轮

司南的壁垒不是单点技术，而是一个随规模自我强化的循环——客户越多，数据与反馈越厚，匹配越准，产品越有价值，再带来更多客户。

🔁

数据飞轮

更多客户 → 更多企业画像 + 匹配反馈 → 匹配越准 → 越有价值 → 更多客户；同时公共层持续沉淀合规结构化招投标数据与历史中标数据，越积越厚。

🏰

四重护城河

① 成本结构（规模越大越省）；② 合规数据资产（时间壁垒，喂养 V3）；③ 只采官方源的合规安全；④ 语义匹配 + 风险模型质量随反馈提升，后来者冷启动难。

资产沉淀：今天采的每一条公告、每一次匹配反馈，都是明天 V3「历史中标分析 / 竞争对手分析」的语料——数据壁垒是买不来、爬不走、只能靠时间积累的。
