# AI 寻标 Agent（Tender Intelligence Agent）

> 开发代号：**Project Compass（司南）**

一套基于人工智能的招标商机智能平台。系统自动采集多个招标采购平台的信息，利用大模型完成信息提取、智能分类、企业能力匹配、商机推荐与风险分析，帮助企业从"人工找项目"升级为"AI 主动发现商机"。

## 产品定位

AI 寻标 Agent 不是传统的招标信息聚合网站，而是一名帮助企业寻找商机的 **AI 智能销售助手**。它替代销售人员每天浏览大量招标网站、人工筛选项目、判断是否适合投标等重复性工作。

**目标用户：** 系统集成商、软件企业、IT 服务公司、智能化工程公司、安防企业、网络设备供应商、AI 企业、制造业与政府项目销售团队、投标部门。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| **招标信息采集** | 自动采集政府采购、公共资源交易、各省市采购网、医院/高校/企业/能源/电网采购平台及自定义网站，支持定时与全天候运行 |
| **AI 信息解析** | 自动提取项目名称、招标单位、发布/截止时间、预算、行业、地区、产品类别、联系方式、招标要求等；解析 PDF/Word 附件（含 OCR）并结构化 |
| **智能分类** | AI 自动分类（IT 类 / 软件类 / 工程类等），无需人工维护复杂关键词 |
| **企业能力匹配** | 基于企业能力画像（产品、服务、行业、区域、资质、案例、品牌）计算项目匹配度与推荐等级 |
| **商机推荐** | 每日分析全部招标，推荐真正值得关注的高价值项目并分级（★★★★★ ~ ★★★☆☆） |
| **风险分析** | 识别品牌限制、排他条件、特殊资质、预算不足、竞争激烈、废标风险，输出参与建议 |
| **消息通知** | 支持邮件、企业微信、微信、钉钉、飞书、Web 消息；每日推送，重大项目即时提醒 |
| **数据查询** | 关键词/行业/地区/预算/时间/采购单位/项目类型筛选，以及 AI 自然语言搜索 |

## 系统流程

```
招标网站 → 自动采集 → 数据清洗 → 附件解析（PDF/Word + OCR）
  → AI 理解 → 结构化数据 → 企业能力匹配 → 商机评分
  → 消息通知 → 销售跟进
```

## 产品路线图

- **V1（寻标 Agent）**：多平台采集、AI 信息解析、企业能力匹配、商机推荐、消息通知。*（当前目标）*
- **V2（投标 Agent）**：招标文件解析、AI 评分分析、自动生成投标方案、风险识别。
- **V3（商机智能体）**：企业知识图谱、历史中标分析、客户画像、竞争对手分析、商机预测、多 Agent 协同决策。

## 产品愿景

通过持续采集市场信息、理解招标需求、匹配企业能力、推荐高价值项目，构建覆盖"发现商机 — 分析商机 — 辅助投标 — 沉淀知识"的智能化业务平台，帮助企业销售团队实现数字化升级。
